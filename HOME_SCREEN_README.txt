### Change the Purple

### Change the Welcome Text
Find: `Text('Welcome, Reverend')`

### Add More Quick Access Cards
In QuickAccessGrid, add to the `final cards` list:
```dart
QuickAccessCardData(
  icon: Icons.settings,
  title: 'Settings',
  subtitle: 'App preferences',
  color: const Color(0xFFF0F0F0),
  iconColor: const Color(0xFF666666),
),
```

### Change Spacing
Find `SizedBox(height: 28)` and change to any value (40, 50, etc.)
